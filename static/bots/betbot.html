About the Bot

A bot meant for horse race style betting using the currency of pokeverse(A pokemon bot) gold, and betting on games of Call of Duty

Commands

Command Name Command Description
/add Add players to the game
/bet Bet on a member of the game
/end End the current game
/help Show a list of commands
/initiate Start up a game
/stop Stop the current game(with no winner)
