About the Bot

This is a discord bot meant to help people with words. Need to know the definition of a word someone used? No problem! Just run the command /define word to get the definition of a word. Or DM the bot a word directly to discretely find out what a word means!

Commands

Command Name Command Description
DM a word Get the definition of the word you DMed the bot
d!define {word} Get the definition of the specified word
d!synonyms {word} Get a list of synonims for the specified word
d!help Get a list of commands
